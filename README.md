# DoorDash Campus Mode

A student-focused product concept designed to improve food delivery through affordability, faster decisions, and personalized discovery.

## Live Demo

[View Live Project](https://kherakriti-ai.github.io/doordash-campus-mode/)

## Overview

DoorDash Campus Mode is a product concept built for college users who frequently order food but often face:

- Price uncertainty from hidden fees
- Decision fatigue from too many choices
- Difficulty finding affordable meals quickly

This concept explores how a student-first experience could improve conversion, trust, and repeat usage.

## Core Features

### Campus Mode
A dedicated experience tailored for student users and campus lifestyles.

### Budget Mode
A core feature that helps users discover meals by **final delivered price** instead of only menu price.

### Smart Decision Tags
Labels such as:

- Fastest
- Best Value

to simplify choices and reduce browsing time.

## Product Goals

- Improve browse-to-checkout conversion
- Reduce cart abandonment
- Increase repeat order frequency
- Build trust through transparent pricing

## Metrics Considered

- Campus Mode adoption rate
- Browse → checkout conversion
- Cart abandonment rate
- Orders per active user
- Repeat usage rate

## Experimentation Idea

**A/B Test**

Control: Standard restaurant listing page  
Variant: Budget Mode with final delivered pricing shown upfront

Hypothesis: Transparent pricing increases conversion and lowers abandonment.
